# 251CTT054_ThiGiuaKiTKWebsitee
BÁO CÁO QUÁ TRÌNH THỰC HIỆN THIẾT KẾ WEBSITE

1. Công cụ AI đã sử dụng
ChatGPT
2. Các prompt chính đã sử dụng
* Thiết kế website quán cà phê bằng HTML, CSS, JavaScript.
* Tạo menu đồ uống sử dụng CSS Grid.
* Tạo khu vực dịch vụ sử dụng Flexbox.
* Viết chức năng tìm kiếm đồ uống bằng JavaScript.
* Viết chức năng đếm số lượt đặt hàng.
* Chỉnh sửa giao diện responsive cho website.
3. Những phần tự thực hiện
* Tự xây dựng cấu trúc thư mục dự án.
* Tự lựa chọn chủ đề Quán Cà Phê Nhà H.
* Tự tìm kiếm và thay thế hình ảnh đồ uống.
* Tự chỉnh sửa nội dung menu, tin tức và thông tin liên hệ.
* Tự kiểm tra và chạy website trên Visual Studio Code bằng Live Server.
* Tự điều chỉnh màu sắc và bố cục theo ý tưởng cá nhân.
4. Những phần tham khảo AI
* Tìm bố cục trang Web
* Mã CSS cho Grid và Flexbox.
* Chức năng JavaScript tìm kiếm đồ uống.
* Chức năng JavaScript đếm số lượt đặt hàng.
* Một số gợi ý về bố cục giao diện và footer.
5. Khó khăn gặp phải
* Khó khăn trong việc sử dụng CSS Grid và Flexbox.
* Chưa biết cách bố trí giao diện sao cho đẹp và cân đối.
* Khó khăn khi thiết kế giao diện tương thích trên nhiều kích thước màn hình.
